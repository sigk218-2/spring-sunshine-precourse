# 날씨조회 LLM 
 - 주어진 도시 이름을 입력 받아 외부 rest api 를 호출하여 위, 경도 조회 
 - 위, 경도로 도시의 날씨 정보 조회 
 - 설정에 따라 기존 함수를 타거나, llm을 호출하는 함수를 타도록 수정 
 - 권역 단위로 날씨 조회 하는 기능
- 날씨 또는 기온을 기준으로 복장 추천
- 각 요청에 대한 사용량과 비용 추정치 로그로 남김
